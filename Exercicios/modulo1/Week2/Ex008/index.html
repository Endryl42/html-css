Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag b q não é semântico.

Agora temos, nessa frase um termo em destaque usando a tag strong que é semântico.

Itálico / Ênfase

Nessa frase, termo em itálico usando a tag I que não é semântico.

Agora, nesta frase, temos um termo com ênfase usando a tag EM que é semântico.

Obs: utilize o atalho shift+ctrl+p para selecionar uma parte da frase e coloca-la em uma tag.

Textos Marcados

Podemos criar também um texto marcado usando a tag MARK.

OBS: Podemos mudar a cor do texto marcado utilizando o css direto na tag: <mark style="background-color: limegreen;"> ou utilizar o css no head.

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a tag DEL, como a tag BIG usado para colocar o texto grande.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS..

existe também a tag U, porém não é semântico

Texto sobrescrito

Para inserir coisas do tipo x2+3, utilizando a tag SUP.

Texto subscrito

Para inserir coisas do tipo h2o, utilizando a tag SUB.
